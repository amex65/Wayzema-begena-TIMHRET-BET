# ዋይዜማ (WAYZEMA)
### የኢትዮጵያ ኦርቶዶክስ ተዋሕዶ ቤተክርስቲያን የዜማ መዝገብ

እንኳን ወደ ዋይዜማ (WAYZEMA) ፕሮጀክት በደህና መጡ። ይህ ገጽ የቅዱስ ያሬድን የዜማ ጥበብ፣ መንፈሳዊ ድርሰቶችን እና የቤተክርስቲያንን ሥርዓተ ማኅሌት ለምእመናን እና ለተማሪዎች ለማድረስ የተዘጋጀ ነው።

---

## 📋 የፕሮጀክቱ ዋና ይዘቶች

### ፩. ሥርዓተ ዜማ (Liturgical Chants)
* **ግዕዝ፦** መሠረታዊው የዜማ ዓይነት።
* **ዕዝል፦** ለበዓላት የሚቆም ጥልቅ ዜማ።
* **አራራይ፦** ለምስጋና እና ለደስታ የሚዘመር።

### ፪. የቅዱስ ያሬድ መጻሕፍት
* **ድጓ እና ጾመ ድጓ**
* **ዝማሬ እና መዋሥዕት**
* **ምዕራፍ እና አክኑኑ**

### ፫. ትምህርቶች እና ምርምር
* የዜማ ምልክቶች (ቅርጽ) ትርጓሜ።
* የሊቃውንት ምስክርነት እና ታሪክ።
