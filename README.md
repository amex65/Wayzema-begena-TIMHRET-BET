ዋየዜማ የበገናና መንፈሳዊ ጥበባት ማሰልጠኛ
የኢትዮጵያ ኦርቶዶክስ ተዋሕዶ ቤተክርስቲያን ዜማ በሦስት ዋና ዋና ክፍሎች ይከፈላል፦
1. **ግዕዝ፦** ለተራ ቀናትና ለጾም የሚሆን ዜማ።
2. **ዕዝል፦** ለበዓላትና ለደስታ ቀናት የሚሆን ዜማ።
3. **አራራይ፦** ለምስጋናና ለጸሎት የሚሆን ለስላሳ ዜማ።

## ፪. የቅዱስ ያሬድ ድርሰቶች
በዋይዜማ አገልግሎት ውስጥ የሚካተቱ ዋና ዋና ድርሰቶች፦
* **ጾመ ድጓ፦** የጾም ወራት ዜማዎች።
* **ምዕራፍ፦** የዓመቱ የማኅሌትና የቅዳሴ ዜማዎች።
* **ዝማሬና መዋሥዕት፦** ለበዓላትና ለቀብር ሥነ-ሥርዓት የሚዘመሩ።
